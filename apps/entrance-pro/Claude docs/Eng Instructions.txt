## CONTENT SPECIFICATIONS

=== FILE 1: comprehension.data.js ===

Generate 4 original passages. For each passage generate 5 MCQ questions.
Total: 20 questions minimum.

PASSAGE RULES:
- Length: 150-200 words each
- Tone: Formal, informative
- Reading level: Class 11-12 (not too simple, not research-level academic)
- Each passage must be on a DIFFERENT theme from this list:
  Theme A: Indian entrepreneurship and startups
  Theme B: Environmental sustainability and green business
  Theme C: Digital economy and fintech in India
  Theme D: Women in business and corporate leadership

QUESTION DISTRIBUTION PER PASSAGE (5 questions):
  Q1: Main idea / central theme of the passage
  Q2: Vocabulary in context — ask meaning of ONE word used in passage
  Q3: Inference — what can be concluded / implied
  Q4: Author's tone or purpose
  Q5: Direct factual detail retrieval from passage

QUESTION RULES:
- type must be "case-study"
- Each question must include the full passage text in the "passage" field
- Distractors must be plausible — not obviously wrong
- Correct answer must be unambiguously supported by the passage text
- explanation must cite the specific sentence/phrase from the passage 
  that justifies the correct answer

SELF-CHECK BEFORE SAVING:
- Is the correct answer clearly supported by passage text? 
- Are all 3 distractors plausible but clearly wrong on re-reading?
- Does the explanation reference the passage directly?
- Are all 5 question types covered per passage?

=== FILE 2: grammar.data.js ===

Generate 20 grammar MCQ questions across these 8 topics (2-3 per topic):

TOPICS:
  1. Tenses — identify correct tense or complete sentence
  2. Subject-Verb Agreement — singular/plural mismatch detection
  3. Articles — correct use of a / an / the / no article
  4. Prepositions — correct preposition in context
  5. Active and Passive Voice — convert or identify voice
  6. Direct and Indirect Speech — convert or identify
  7. Spotting Errors — one sentence with 4 underlined parts, 
     identify the part with grammatical error
  8. Sentence Rearrangement — 4 jumbled sentences, find correct order

QUESTION TYPE MAPPING:
- Topics 1-6: type = "mcq"
- Topic 7 (Spotting Errors): type = "mcq" 
  (options are A/B/C/D referring to underlined parts)
- Topic 8 (Rearrangement): type = "mcq"
  (options are different ordering sequences like BACD, ABCD, CDAB, DCBA)

GRAMMAR QUESTION RULES:
- Each question must test ONE specific grammar rule — stated in explanation
- Name the grammar rule being tested in the explanation field
- Distractors must represent common student errors, not random wrong answers
- Difficulty: 10 easy + 7 medium + 3 hard
- All sentences should use commerce/business/general knowledge themes
  (avoid trivial sentences like "Ram went to school")

EXAMPLE of a good distractor strategy for Subject-Verb Agreement:
  Sentence: "The quality of the products __ very poor."
  Correct: "is" (subject = quality, singular)
  Distractor 1: "are" (common error — student looks at "products" not "quality")
  Distractor 2: "were" (tense confusion)
  Distractor 3: "have been" (aspect confusion)

SELF-CHECK BEFORE SAVING:
- Is exactly ONE option correct with no ambiguity?
- Does the explanation name the specific grammar rule?
- Do distractors represent real student errors?
- Are all 8 grammar topics represented?

=== FILE 3: vocabulary.data.js ===

Select 10 words from the SEED LIST below.
For each word, generate a FAMILY of 4 questions = 40 questions total.

SEED WORD LIST (pick 10):
  Prudent, Incentive, Volatile, Deficit, Surplus, Arbitrary,
  Autonomy, Lucrative, Pragmatic, Eloquent, Ambiguous, Viable,
  Consensus, Cumulative, Leverage, Resilient, Complacent,
  Diligent, Formidable, Concise

QUESTION FAMILY PER WORD (4 questions):

  Q-TYPE A (Direct Meaning):
  "The word [WORD] in the sentence '[USE WORD IN A SENTENCE]' 
   most nearly means:"
  Options: correct synonym + 3 plausible distractors

  Q-TYPE B (Synonym):
  "Which of the following is closest in meaning to [WORD]?"
  Options: correct synonym + 3 near-synonyms that are subtly different

  Q-TYPE C (Antonym):
  "Which of the following is OPPOSITE in meaning to [WORD]?"
  Options: correct antonym + 3 distractors

  Q-TYPE D (Fill in the Blank):
  "The CEO was known for her _____ approach — she never made 
   decisions without careful analysis of data."
  (Blank should be filled by the target word)
  Options: target word + 3 contextually plausible but incorrect words

VOCABULARY RULES:
- All sentences must use professional/business context
- For Q-TYPE B (Synonym), distractors should be near-synonyms 
  (words that are related but not equivalent) — not random words
- difficulty: 
    Q-TYPE A and D = "easy"
    Q-TYPE B = "medium"  
    Q-TYPE C = "medium"
- explanation must define the word AND explain why distractors are wrong

SELF-CHECK BEFORE SAVING:
- Is the Fill-in-the-blank sentence natural and unambiguous?
- Are synonym distractors genuinely close but distinguishably wrong?
- Is the antonym unambiguous?
- Does explanation define the word clearly?

## QUALITY GATES (Agent must verify before outputting each file)

GATE 1 — CONTENT QUALITY:
  [ ] Every passage is original (not copied from any source)
  [ ] Passages are factually accurate (no false claims about India/economy)
  [ ] No culturally insensitive, religious, or political content
  [ ] No personally identifiable or real individual's name used as subject
      (use fictional names like "Priya", "Arjun", "Meera" for examples)

GATE 2 — QUESTION QUALITY:
  [ ] Every question has exactly ONE unambiguously correct answer
  [ ] No trick questions — correct answer is fair and findable
  [ ] Explanations are complete and educational (teach, don't just confirm)
  [ ] Distractors represent real student errors/misconceptions

GATE 3 — SCHEMA COMPLIANCE:
  [ ] All required fields present per DATA_SCHEMA.md
  [ ] correct field is a valid index (0, 1, 2, or 3)
  [ ] options array has exactly 4 items
  [ ] examTags is ["MH-CET"] for all English questions
  [ ] IDs are unique within the file (comp-q-001 through comp-q-020)
  [ ] Minimum question counts met: comp=20, grammar=20, vocab=40

GATE 4 — LANGUAGE QUALITY:
  [ ] No grammatical errors in question stems or options
  [ ] No spelling mistakes
  [ ] Consistent Indian English conventions (not American English)
  [ ] Passage sentences are varied in length (not all the same structure)

## ANTI-HALLUCINATION RULES FOR THIS AGENT

1. PASSAGES: Do not reference real news events, real companies by name, 
   or real statistics unless they are well-established facts 
   (e.g. "India is the world's most populous country" is fine; 
   "Company X reported 40% growth in 2024" is not)

2. GRAMMAR: Every grammar rule cited in explanation must be a real, 
   named grammar rule. Do not invent rule names.

3. VOCABULARY: Every word meaning, synonym, and antonym must be 
   verifiable in a standard dictionary. Do not use obscure or 
   contested meanings.

4. NEVER invent question IDs that conflict with other data files.
   English file IDs use prefixes: comp-, gram-, vocab-

5. If uncertain about the correct answer to your own question, 
   FLAG IT with a comment:
   // REVIEW NEEDED: answer may be ambiguous
   Do not silently include a questionable question.

6. If the JSON structure becomes complex, output it in sections 
   and confirm structure before proceeding to content.
