## UPDATED SCHEMA FOR CA QUESTIONS

Every CA question must now include:

  caTag: "national-polity" | "international" |
         "business-economy" | "science-defence" |
         "awards-sports"

  caYear: 2023 | 2024 | 2025 | "static"
    // Allows app to refresh only recent year questions

  verifiedAsOf: "YYYY-MM"
    // For any fact that could change

  refreshPriority: "high" | "medium" | "low"
    // high = appointment/rate (changes often)
    // medium = award/event (annual cycle)
    // low = founding year/static fact (never changes)

Example question object:
  {
    id: "ca-np-001",
    type: "mcq",
    difficulty: "easy",
    question: "Who is the Chairman of the 16th Finance Commission?",
    options: [
      "Dr. Arvind Panagariya",
