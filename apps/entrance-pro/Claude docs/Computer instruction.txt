--- TOPIC 2: Memory Hierarchy (15 questions) ---

MEMORY TYPES — Master List:
  RAM (Random Access Memory):
    → Volatile (data lost on power off)
    → Primary memory, directly accessed by CPU
    → Types: DRAM, SRAM
  ROM (Read Only Memory):
    → Non-volatile (data retained after power off)
    → Stores BIOS/firmware
    → Types: PROM, EPROM, EEPROM
  Cache Memory:
    → Fastest memory in hierarchy
    → Sits between CPU and RAM
    → L1 (fastest, smallest), L2, L3 levels
  Secondary Storage:
    → Non-volatile: HDD, SSD, CD, DVD, USB Drive
    → SSD: faster than HDD, no moving parts
  Virtual Memory:
    → Part of secondary storage used as RAM extension
    → Slower than actual RAM
  Registers:
    → Inside CPU, fastest and smallest
    → Hold data currently being processed

MEMORY HIERARCHY (slowest to fastest):
  Secondary → RAM → Cache → Registers

QUESTION PATTERNS:

  Volatile vs Non-volatile (5 questions):
    → "Which memory loses data when power is switched off?"
      (RAM — Volatile)
    → "Which of the following is NON-volatile memory?"
      (ROM / SSD / HDD — all non-volatile)
    → "BIOS is stored in which type of memory?" (ROM)
    → Trap: "Is cache volatile or non-volatile?" (Volatile)
    → "Which memory retains data without power supply?"

  Memory Speed Hierarchy (5 questions):
    → "Which is the fastest memory in a computer?" (Cache/Registers)
    → "Arrange in order of speed: RAM, Cache, HDD, Registers"
    → "Which memory acts as a bridge between CPU and RAM?"
      (Cache)
    → "SSD is faster than HDD because..."
      (No moving parts, electronic storage)
    → "Virtual memory is actually part of..." (Secondary storage)

  Identification Template (5 questions):
    → "Which of the following is PRIMARY memory?"
      (RAM — not HDD, not SSD, not DVD)
    → "Which is NOT a type of ROM?"
      (DRAM — that is RAM, not ROM)
    → "Which storage device has NO moving parts?"
      (SSD — correct; HDD has spinning disk)
    → "Which memory is directly accessed by the CPU?"
      (RAM / Cache / Registers — primary memory)
    → "Which of the following is used to store the
      operating system permanently?" (ROM / SSD)

--- TOPIC 3: CPU Architecture (15 questions) ---

CPU COMPONENTS — Master List:
  ALU (Arithmetic Logic Unit):
    → Performs arithmetic: +, –, ×, ÷
    → Performs logical comparisons: AND, OR, NOT, >, <, =
    → This is THE most tested CPU component in MAH CET
  CU (Control Unit):
    → Directs operations of the processor
    → Does NOT process data — only coordinates
    → Fetches, decodes and executes instructions
  MU (Memory Unit):
    → Temporary storage inside CPU
  Registers:
    → Small, ultra-fast storage locations inside CPU
    → Types: Accumulator, Program Counter, Instruction Register
  Buses:
    → Data Bus: carries data between components
    → Address Bus: carries memory addresses
    → Control Bus: carries control signals

QUESTION PATTERNS:

  ALU Questions (6 questions) — HIGHEST PRIORITY:
    → "Which unit of the CPU performs logical comparisons?"
      (ALU) — this is the #1 most tested question
    → "Which part of the CPU performs arithmetic operations?"
      (ALU)
    → "ALU stands for..." (Arithmetic Logic Unit)
    → "Which CPU unit executes calculations like addition
      and subtraction?" (ALU)
    → "What does the 'L' in ALU stand for?" (Logic)
    → "Which of the following is NOT a function of ALU?"
      (Fetching instructions — that is CU's job)

  CU vs ALU Distinction (5 questions):
    → "Which CPU unit coordinates and directs all
      computer operations?" (CU — not ALU)
    → "Decoding of instructions is done by..." (CU)
    → "Which unit does NOT process data but controls
      the flow of data?" (CU)
    → "The fetch-decode-execute cycle is managed by..." (CU)
    → Match: ALU ↔ calculations; CU ↔ coordination

  Registers and Bus (4 questions):
    → "Which register stores the address of the next
      instruction to be executed?" (Program Counter)
    → "Which bus carries data between CPU and memory?"
      (Data Bus)
    → "Accumulator is a type of..." (Register)
    → "Which is the fastest storage inside a computer?"
      (Registers — faster than cache)

HARDWARE FLASHCARDS (15 flashcards):
  - Input devices complete list
  - Output devices complete list
  - Devices that are BOTH input and output
  - RAM: volatile, primary, CPU-accessible
  - ROM: non-volatile, stores BIOS
  - Memory speed order: Registers > Cache > RAM > Secondary
  - Cache: bridge between CPU and RAM
  - SSD vs HDD: no moving parts vs spinning disk
  - Virtual memory: secondary storage used as RAM
  - ALU: arithmetic + logical operations
  - CU: fetch-decode-execute, coordinates operations
  - Registers: fastest, smallest, inside CPU
  - Data Bus vs Address Bus vs Control Bus
  - Volatile memory: RAM, Cache (lose data on power off)
  - Non-volatile: ROM, SSD, HDD (retain data)

  [ ] Touchscreen tagged as BOTH input and output

--- TOPIC 1: Internet Protocols (15 questions) ---

PROTOCOLS — Master List:
  HTTP  (HyperText Transfer Protocol):
    → Used for transferring web pages
    → NOT secure — data is unencrypted
  HTTPS (HyperText Transfer Protocol Secure):
    → Secure version of HTTP
    → Uses SSL/TLS encryption
    → Bank websites, login pages use HTTPS
  FTP   (File Transfer Protocol):
    → Used for transferring files between computers
    → NOT for web browsing
  SMTP  (Simple Mail Transfer Protocol):
    → Used for SENDING emails
  POP3  (Post Office Protocol 3):
    → Used for RECEIVING emails (downloads to device)
  IMAP  (Internet Message Access Protocol):
    → Used for RECEIVING emails (stays on server)
  TCP/IP (Transmission Control Protocol/Internet Protocol):
    → Foundation of internet communication
    → TCP ensures data delivery; IP handles addressing
  DNS   (Domain Name System):
    → Converts domain names to IP addresses
    → "Phone book of the internet"
  IP Address:
    → Unique address of device on network
    → IPv4: 4 sets of numbers (192.168.1.1)
    → IPv6: newer, longer format
  URL   (Uniform Resource Locator):
    → Complete web address (https://www.google.com)
  WWW   (World Wide Web):
    → Collection of websites/web pages
    → NOT the same as the internet
    → Invented by Tim Berners-Lee (1989)

QUESTION PATTERNS (15 questions):

  Protocol identification (6 questions):
    → "Which protocol is used for SECURE data transfer
      on the web?" (HTTPS)
    → "FTP is primarily used for..." (File transfer)
    → "Which protocol converts domain names to IP addresses?"
      (DNS)
    → "SMTP is used for..." (Sending emails)
    → "What is the difference between HTTP and HTTPS?"
      (HTTPS is encrypted/secure)
    → "Which protocol is used to RECEIVE emails and
      keeps them on the server?" (IMAP — not POP3)

  WWW and Internet distinction (4 questions):
    → "WWW was invented by..." (Tim Berners-Lee)
    → "Which of the following is NOT the same as
      the Internet?" (WWW — subset of internet)
    → "URL stands for..." (Uniform Resource Locator)
    → "What does the 'S' in HTTPS stand for?" (Secure)

  TCP/IP and addressing (5 questions):
    → "TCP/IP stands for..."
      (Transmission Control Protocol/Internet Protocol)
    → "An IP address uniquely identifies..."
      (a device on a network)
    → "IPv4 addresses consist of how many sets of numbers?"
      (4 sets — e.g., 192.168.1.1)
    → "Which layer of TCP/IP handles logical addressing?"
      (IP — Internet Protocol layer)
    → "DNS is called the 'phone book of the internet'

--- TOPIC 2: Networking Basics (15 questions) ---

NETWORK TYPES — Master List:
  LAN (Local Area Network):
    → Small geographic area: office, school, home
    → High speed, privately owned
  MAN (Metropolitan Area Network):
    → City-wide network
    → Connects multiple LANs
  WAN (Wide Area Network):
    → Large geographic area: countries, continents
    → Internet is the largest WAN
  PAN (Personal Area Network):
    → Very short range: Bluetooth devices
  WLAN (Wireless LAN):
    → WiFi network

NETWORK TRANSMISSION TYPES:
  Unicast:   One sender → One receiver
  Broadcast: One sender → ALL receivers on network
  Multicast: One sender → SPECIFIC GROUP of receivers
    → "Multicasting = sending to a group of stations"
    → This exact definition appears in MAH CET papers

NETWORK DEVICES:
  Router:  Connects different networks, directs traffic
  Switch:  Connects devices within same network (smarter than hub)
  Hub:     Connects devices, sends to ALL (dumb device)
  Modem:   Converts digital ↔ analog signals (for internet)
  Bridge:  Connects two similar networks
  Gateway: Connects two different types of networks
  Repeater: Boosts signal over long distances

TOPOLOGY TYPES:
  Star Topology:   All devices connect to central hub/switch
  Bus Topology:    All devices on single cable
  Ring Topology:   Devices in circular connection
  Mesh Topology:   Every device connected to every other

--- TOPIC 1: Malware and Threats (15 questions) ---

MALWARE TYPES — Master List:
  Virus:
    → Attaches to files, spreads when file is shared
    → Requires human action to spread
    → Can corrupt/delete files
  Worm:
    → Self-replicating, spreads WITHOUT human action
    → Travels through networks automatically
    → Key distinction from virus: NO human trigger needed
  Trojan Horse:
    → Disguises as legitimate software
    → Does NOT self-replicate
    → Creates backdoor for attacker
  Spyware:
    → Secretly monitors user activity
    → Collects passwords, browsing history
    → User does NOT know it is running
  Ransomware:
    → Encrypts user's files
    → Demands payment (ransom) to restore access
    → WannaCry is a famous example
  Adware:
    → Displays unwanted advertisements
    → Often bundled with free software
  Keylogger:
    → Records every keystroke
    → Used to steal passwords and credit card numbers
  Rootkit:
    → Hides deep in OS, gives attacker admin access
    → Very difficult to detect and remove
  Phishing:
    → Fake emails/websites to steal credentials
    → Social engineering attack (not a program)
  Denial of Service (DoS):
    → Floods server with requests to make it crash

QUESTION PATTERNS (15 questions):

  Malware identification (6 questions):

--- TOPIC 2: Security Tools and Safe Practices (15 questions) ---

SECURITY CONCEPTS:
  Firewall:
    → Monitors and filters incoming/outgoing network traffic
    → Blocks unauthorised access
    → Can be hardware OR software
    → Does NOT remove existing viruses
      (that is antivirus's job)
  Antivirus:
    → Detects and removes malware
    → Must be updated regularly
    → Examples: Norton, Kaspersky, Windows Defender
  Encryption:
    → Converts data to unreadable format
    → Decryption reverses the process
    → HTTPS uses encryption
  SSL/TLS:
    → Protocols for encrypted communication
    → Padlock symbol in browser = SSL active
  Two-Factor Authentication (2FA):
    → Two verification steps to login
    → Password + OTP (One Time Password)
  VPN (Virtual Private Network):
    → Creates encrypted tunnel for internet traffic
    → Hides user's IP address
  Backup:
    → Copy of data for recovery after attack/failure
    → Best practice: 3-2-1 rule
      (3 copies, 2 different media, 1 offsite)
  Password Safety:
    → Strong: 8+ characters, mixed case, numbers, symbols
    → Never share, reuse, or write down

QUESTION PATTERNS (15 questions):

  Firewall questions (4 questions):
    → "What is the PRIMARY function of a firewall?"
      (Monitor and filter network traffic)
    → "A firewall can be..." (hardware OR software — both)
    → "Which of the following is NOT done by a firewall?"
      (Removing existing viruses — that is antivirus)
    → "Firewall protects against..."
      (unauthorised network access)

  Encryption (4 questions):
    → "HTTPS uses which security protocol?"
      (SSL/TLS)
    → "What does a padlock icon in a browser indicate?"
      (SSL/TLS encryption is active — secure connection)
    → "Encryption converts data into..."
      (unreadable/ciphertext format)
    → "2FA stands for..." (Two-Factor Authentication)

  Safe practices (4 questions):
    → "VPN primarily helps by..."
      (encrypting internet traffic and hiding IP address)
    → "Which is the BEST practice for password security?"
      (Use unique, complex password for each account)
    → "Regular antivirus updates are important because..."
      (new malware is released daily; old definitions miss it)
    → "2FA adds security by requiring..."
      (a second verification step beyond password)

  General identification (3 questions):
    → "Which of the following is a SECURITY tool
      (not a threat)?"
      Options: Worm, Spyware, Firewall, Ransomware
      (Firewall)
    → "SSL stands for..."
      (Secure Sockets Layer)
    → "The best defence against Ransomware is..."
      (regular data backups)

CYBERSECURITY FLASHCARDS (10 flashcards):
  - Virus vs Worm: human trigger vs automatic spread
  - Trojan: disguised, no self-replication, creates backdoor
  - Ransomware: encrypts files, demands payment
  - Spyware: silent monitor; Keylogger: records keystrokes
  - Phishing: fake emails/sites — social engineering
  - Firewall: filters traffic; does NOT remove viruses
  - Antivirus: detects and removes malware
  - SSL/TLS: encryption for web; padlock = active
  - 2FA: two verification steps
  - VPN: encrypted tunnel, hides IP address

NUMBER SYSTEMS — Master Reference:

  Binary (Base-2):    Digits: 0, 1
  Octal (Base-8):     Digits: 0–7
  Decimal (Base-10):  Digits: 0–9  (normal numbers)
  Hexadecimal (Base-16): Digits: 0–9 and A(10) B(11) C(12)
                         D(13) E(14) F(15)

CONVERSION METHODS (state these in solution_steps):

  Binary → Decimal:
    → Multiply each bit by 2^position, sum all
    → 1011₂ = 1×8 + 0×4 + 1×2 + 1×1 = 11₁₀

  Decimal → Binary:
    → Divide by 2 repeatedly, read remainders bottom-up
    → 13₁₀ = 1101₂

  Hexadecimal → Decimal:
    → Multiply each digit by 16^position
    → 1F₁₆ = 1×16 + 15×1 = 31₁₀

  Decimal → Hexadecimal:
    → Divide by 16, use A-F for remainders 10-15

  Binary → Hexadecimal:
    → Group binary digits in sets of 4 from right
    → Convert each group to hex digit
    → 11110001₂ = F1₁₆
