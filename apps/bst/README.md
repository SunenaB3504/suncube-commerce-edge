# CBSE Class 12 Business Studies - Exam Prep App

A premium, interactive learning application for CBSE Class 12 Business Studies, featuring:

*   **Theory Master**: Deep dive into management principles (Units 1-12).
*   **Visual Learning**: Hierarchical breakdowns with Purple theme.
*   **Revision HQ**: Flashcards, Mind Maps, and Cheat Sheets.
*   **SQP Archives**: Verified Board Questions with Marking Schemes.

## Tech Stack
*   React 19
*   TypeScript
*   TailwindCSS (Purple Theme)
*   Vite
*   Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```
